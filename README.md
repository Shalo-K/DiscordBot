# DiscordBot

## 概要
Discordサーバ向けのBotです。

自己学習を目的として、プリンセスコネクトのクランバトルでプレイヤー間の情報連携をする補助ツールとして開発していました。

Botとしての主な機能は、Botメッセージへのリアクションを起点にユーザ情報を追加/取消をしたり、記録用のメッセージを専用チャンネルに投稿したりします。

また、記録用のメッセージを集計してcsvで出力(DM送付)させることも可能です。

今後は汎用的な新機能の実装を検討しています。

実装ライブラリはdiscord.pyですが、実行環境にDocker Composeを採用しているため、Dockerコンテナが使用できる環境であればどこでも動作可能です。

Cogを用いてなるべく機能単位で実装するようにしており、必要な機能だけを入れた状態で利用できる構成となっています。<br>
※テスト時には全機能を載せた状態のみ確認しています。

## 動作環境
* Python 3.12
* Docker
* Docker Compose
